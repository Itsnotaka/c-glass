## What Changed

- **Shell / workbench:** Adds workbench tab strip and panel wiring, plus a terminal panel surface integrated into the glass shell (`workbench-tabs`, `workbench-panel`, `terminal/panel`). Shell layout hooks (`app.tsx`, `chat.tsx`, `use-glass-shell-panels`) are updated to support the new panels.
- **Git panel:** Refactors diff loading to **TanStack Query** via `glass-git-react-query.ts` and `use-glass-git.ts`, with tests. Adds a **commit dialog** component and refreshes the git panel and diff viewer behavior.
- **Chat / markdown:** Streamdown chat markdown uses **Pierre Display P3** Shiki themes (`@pierre/theme` vibrant imports in `chat-streamdown.ts`) and keeps the plugin/theme wiring in `chat-markdown.tsx`. Chat rows, timeline, and provider notice helpers are adjusted for consistency.
- **Server:** Small updates to `ProviderRuntimeIngestion.ts` plus a new **unit test** file `ProviderRuntimeIngestion.test.ts`.
- **Tooling:** `pnpm-lock.yaml`, `package.json`, `main.tsx`, and `tailwind.css` updates for new dependencies and styles.

## Why

Centralizes git file patches behind React Query for caching, deduplication, and simpler expand/collapse flows. Workbench and terminal panels move the product toward an IDE-like shell. Pierre vibrant themes align code blocks in chat with the wide-gamut Pierre editor themes. Provider ingestion tests and tweaks reduce regression risk around runtime event handling.

## UI Changes

Workbench tabs, terminal panel, git panel (including commit flow), and chat code-block theming are visually affected. **Screenshots and video not attached in this draft;** reviewers should run `pnpm run dev:web` (or full `dev`) locally to verify layout and git flows.
